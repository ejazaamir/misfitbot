MisfitBot Command Handbook

Full list of slash commands, reply flows, and mention syntax. Commands marked with ADMIN only require elevated permissions in this server.

Quick Permission Legend

Everyone ADMIN only

How To Use

Slash Commands

Type / in chat, choose a command, fill required options, submit.

Reply Workflows

Reply to a specific message first, then use the matching command or mention trigger.

Right Click Apps

Right click a message, open Apps, run Misfit context actions instantly.

Quiz + MBTI + Profiles

Quiz

  • /quiz leaderboard [limit] - everyone
  • /quiz start, /quiz skip, /quiz stop, /quiz clearleaderboard - ADMIN only
  • !hint in quiz channel for hint

MBTI + Profiles

  • /mbti start, /mbti result - everyone
  • /mbti reset user:@User - ADMIN only
  • /profile set|show|clear - everyone
  • /profile peek|setfor|clearfor - ADMIN only

Enable This As GitHub Website

  1. Push this repo to GitHub.
  2. Open repository Settings -> Pages.
  3. Under Build and deployment, set Source to Deploy from a branch.
  4. Choose branch main and folder /docs.
  5. Save. GitHub will publish your site in about 1-2 minutes.
